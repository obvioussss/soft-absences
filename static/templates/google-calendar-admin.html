Configuration et documentation
Configuration

Pour configurer l'intégration Google Calendar :

  1. Consultez le fichier GOOGLE_CALENDAR_SETUP.md
  2. Créez un Service Account sur Google Cloud Console
  3. Activez l'API Google Calendar
  4. Partagez le calendrier avec le Service Account
  5. Configurez la variable d'environnement GOOGLE_CALENDAR_CREDENTIALS
Fonctionnalités

L'intégration Google Calendar :

  • Crée automatiquement des événements pour les demandes approuvées
  • Met à jour les événements si les demandes sont modifiées
  • Supprime les événements si les demandes sont refusées
  • Affiche le nom de l'employé et le type d'absence
  • Utilise des couleurs différentes selon le type d'absence
Conseil : Utilisez le bouton "Tester l'intégration" pour vérifier que tout fonctionne correctement après la configuration.
